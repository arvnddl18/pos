# Printing

Safari on iPad cannot reliably drive USB thermal printers. Options:

1. **Network ESC/POS printer** on the shop LAN; use a small **bridge** (Raspberry Pi or PC agent) that accepts HTTPS or WebSocket print jobs from your network.
2. **Email/cloud print** if the printer vendor supports it.
3. **Receipt as PDF/SMS/email** from the POS (future channel).

Keep a paper backup path (manual receipt pad) if Wi‑Fi fails.
